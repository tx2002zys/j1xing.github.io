001 - Yokan Party（★4）

题目链接

题目描述

有一个长度为L[cm]的木棍。有N个标记点，第i个标记点距左端点的长度为$A_{i}$[cm]。
你要从N个标记点中选择K个，将木棍分成K+1块。K+1块中最短的一块的长度为得分。
找出划分木棍时得到的分数，使分数达到最大。

限制条件
$ 1 ≤ k ≤ n ≤ 10^{5}$
$0 < a_{1} < a_{2}< … < L ≦ 10^{9}$所有的输入都是整数。

二分(最小值最大化)

编辑
atcoder
